## Labs 清单 (CS:APP 第3版)

| # | 目录 | Lab 名称 | 对应章节 | 说明 |
|---|------|----------|----------|------|
| 1 | datalab-handout | Data Lab | 第2章 信息的表示和处理 | 位操作练习 |
| 2 | bomb | Bomb Lab | 第3章 程序的机器级表示 | 逆向工程拆炸弹 |
| 3 | target1 | Attack Lab | 第3章 程序的机器级表示 | 缓冲区溢出攻击 |
| 4 | archlab-handout | Architecture Lab | 第4章 处理器体系结构 | Y86-64流水线优化 |
| 5 | cachelab-handout | Cache Lab | 第6章 存储器层次结构 | 缓存模拟器与矩阵转置优化 |
| 6 | shlab-handout | Shell Lab | 第8章 异常控制流 | 实现简易Unix Shell |
| 7 | malloclab-handout | Malloc Lab | 第9章 虚拟内存 | 实现动态内存分配器 |
| 8 | proxylab-handout | Proxy Lab | 第10-12章 网络编程/并发 | 实现并发Web代理 |

> **注意:** `perflab-handout` 是 CS:APP **第2版**的 Performance Lab，在第3版中已被 Cache Lab 取代，可忽略。
